Welcome Testers!

Hi! Welcome to my second game. This game was created to implement my latest algorithm to solve dense graph problem. It is a simple PUZZLE Game where all switches able to affect others. Try to solve it by switching ON all of them.
Your responsibility is to give feedbacks and recommendations for this game.

For further information about the theory, please don't hesitate to contact me through email : walf.redo.studio@gmail.com

Features in Complete Game

Below are some suggestions from feedbacks for features in complete game:
(+) Daily Missions
(+) Coins that can be receieved once solve a puzzle
(+) A shop that use those coins
(+) Grid Menu for levels

Development

The game is pratically simple. Currently, this prototype is being developed with HTML5 + CSS3 and Javascript. It is uploaded with firebase server because it is free and cross-platform. The complete game will be developed even further with Unity3D. There will be more levels, animations, challenges, achievements, and a simple shop. Stay tuned!
