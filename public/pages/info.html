Welcome Testers!

Hi! Welcome to my second game. This game was created to implement my latest algorithm to solve dense graph problem. It is a simple PUZZLE Game where all switches able to affect others. Try to solve it by switching ON all of them.
Your responsibility is to give feedbacks and recommendations for this game.

For further information about the theory, please don't hesitate to contact me through email : walf.redo.studio@gmail.com

Tips and Tricks

Some testers said "The puzzle is too hard!" , "All looks random for me", "It is a trial and error game, I can solve it without thinking.", etc... Thus, here is some tips from me:

(-)   Find the pattern. The pattern is from the most influence switch to the switch that not affect others.
(-)   Draw it on paper, and you might understand the logic of the game.
(-)   Try to not do this randomly, because I don't make it random :p . What is the fun of puzzle game if you do it randomly?

Development

The game is pratically simple. Currently, this game is being developed with HTML5 + CSS3 and Javascript. It is uploaded with firebase server because it is free and cross-platform. The ads also shows up in the first page (sorry if it annoys you). The complete game will be developed even further with Unity3D. There will be more levels, animations, challenges, achievements, and a simple shop. Stay tuned!
